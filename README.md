# CHAT-AI

Descrição breve do que o projeto faz.

## Funcionalidades
- Conversa Com iA (INTEGRAÇÂO COM O GEMINI)
- Salva Historico de mensagem no banco de dados

## Tecnologias
- Java
- Spring Boot
- PostgreSQL
- Docker
- API Gemini

## Instalação

1. Clone o repositório

git clone [https://github.com/seuusuario/seuprojeto](https://github.com/DYUCOR/Chat-AI.git)

2. Rode o docker
docker compose up -d

3. Rode o projeto e acesse na porta 8080/index.html
